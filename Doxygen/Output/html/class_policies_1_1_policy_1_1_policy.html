Detailed Description

The Policy determines the action that an Agent will take given its Representation.

The Agent learns about the Domain as the two interact. Each step, the Agent passes information about its current state and information revelent to that state to the Policy. The Policy uses this information to decide what action the Agent should perform next.

The Policy class is a superclass that provides the basic framework for all policiess. It provides the methods and attributes that allow child classes to interact with the Agent and Represemtatopm classes within the RL-Python library.
All new policty implimentations should inherit from Policy.
