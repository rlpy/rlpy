Detailed Description

Developed by Alborz Geramiard Oct 25th 2012 at MIT #.

State is x,y, Actions are 4 way directional with fixed noise each grid cell is: 0: empty 1: blocked 2: start 3: goal 4: pit The task is to reach the goal from the start while avoiding the pits
