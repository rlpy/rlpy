def batchDiscover (   self,
  td_errors,
  all_phi_s,
  s 
)

Adds new features based on the Bellman Error in batch setting.

Parameters
td_errors,:p-by-1 (How much error observed for each sample)
all_phi_s,:n-by-p features corresponding to all samples (each column corresponds to one sample)
s,:List of states corresponding to each td_error in td_errors (note that the same state may appear multiple times because of different actions taken while there) self.batchThreshold is threshold below which no more BEBFs are added.
